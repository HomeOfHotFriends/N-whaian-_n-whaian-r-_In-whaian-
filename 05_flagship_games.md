
# 05 — Flagship Games

Each flagship is not just a game. It is a machine-event.

A place where the console reveals one of its deepest laws in a form you can feel in your hands.

| Game | What it proves |
|------|---------------|
| House of Crash | pressure = composition |
| Space Dead & Herman | dual truths are playable |
| Sine Square Root | game = instrument |

These five are not a catalogue. They are a proof set.

You do not need lots of games. You need a few games that reveal the whole machine.

## 1. House of Crash

**Pressure becomes composition**

### What it is

An isometric destruction-navigation game where you pilot a medium bulldozer through domestic architecture. Every contact reshapes:

- space (routes open/close)
- matter (debris, supports, collapse)
- sound (acoustics, resonance, noise)
- future possibility (what can still hold, what must give)

There is no attack button. There is only movement as pressure.

### Core logic connection

This is the purest embodiment of continuous values plus threshold emergence.

- no discrete damage states
- no fixed verbs like smash / don't smash
- everything is: strain, fatigue, angle, contact duration, accumulated history

And crucially: no event changes only one thing.

Every push is:

- route editing
- acoustic editing
- score shaping
- memory writing

### Mechanics that matter

- Bulldozer body values (traction, torque, blade height, wobble)
- Material gospel (wood ≠ plaster ≠ tile ≠ pipe)
- Support vs surface distinction
- Floor as threshold (not just ground)
- Exterior breach = weather invasion
- Score as elegance of destruction, not avoidance

### What it opens for the ecosystem

- audio-to-structure and structure-to-audio cartridges become real
- Crushhouse becomes a general-purpose transformation engine
- material systems become reusable across games
- "pressure" becomes a universal modulation source

This is where material thinking becomes exportable.

### The respect barrier

If you treat it as: `"haha smash house"` — you get stuck, lost, frustrated.

If you treat it as: a system of pressures you can learn — it opens completely.

### Why it's special

Most destruction games are: spectacle + reset.

This is: interpretation + consequence.

You're not trying to destroy everything. You're trying to read a house while breaking it.

## 2. Space Dead & Herman

**Two truths, one body**

### What it is

A shmup-like game where you control two entities simultaneously:

- **Space Dead** — likely more abstract, spatial, systemic
- **Herman** — grounded, embodied, reactive

Not two lives. Not two modes. Two coexisting operational truths.

### Core logic connection

This is the playable version of dual operand plane architecture.

```
Core A  =  Space Dead logic
Core B  =  Herman logic
```

And the player is Big Friendly: mediating, prioritizing, collapsing, switching emphasis.

### Mechanics that matter

- shared input, split consequence
- position vs field control tension
- co-dependence without symmetry
- collapse moments (temporary unity)
- drift between the two instead of hard switching

### What makes it special

Most dual-character games do: switch between characters.

This does: hold two logics at once.

That is fundamentally different.

### The respect barrier

If you try to master one side: you lose.

If you learn to inhabit both imperfectly: you win.

### Why it's special

The machine's deepest law made playable: two truths are both real, always.

Now it's under your thumbs.

## 3. Sine Square Root

**The in-between made playable**

### What it is

A game where:

- sound is structure
- structure is rule
- rule is motion
- motion is composition

Not metaphorically. Mechanically.

### Core logic connection

This is the fusion of signal flow, game logic, and continuous motion systems.

It proves: game = instrument = system.

### Mechanics that matter

- movement generates sound
- sound feeds rule conditions
- timing affects geometry
- parameter motion changes playfield behavior
- score = compositional clarity

### What it opens

- full legitimacy for hybrid cartridges
- audio-to-game translation pipelines
- tool/game boundary dissolves cleanly
- Tracker of Intent becomes universal

### What makes it special

Most music games: react to music.

This: is music.

### The respect barrier

If you want passive rhythm reaction: you'll hate it.

If you engage as a composer inside a system: you'll get obsessed.

### Why it's special

It is the cleanest answer to: "what if the machine actually means what it says?"

## 4. [Flagship Deterministic Shmup]

**Authored infinity**

### What it is

A fully deterministic shmup where:

- everything is authored
- nothing is random
- depth comes from interaction, not variation

### Core logic connection

This is atomic design plus emergent interaction:

- simple enemy rules
- simple formation bodies
- simple path bones
- complex results through collision

### Mechanics that matter

- formation cohesion
- path segment behavior
- event bones
- field objects (scarecrow et al.)
- score as interpretation
- point-blank as pressure transfer

### What it opens

- reusable formation systems
- wave composition tools
- score-driven world interaction
- exportable enemy logic to other cartridges

### What makes it special

It rejects procedural novelty. It builds depth through understanding.

### The respect barrier

If you want chaos to hide your mistakes: you'll hate it.

If you want clarity that rewards attention: you'll love it.

### Why it's special

Truxton + DoDonPachi energy filtered through system thinking.


---

## 5. [Flagship Hub-World Game]

**The house that remembers**

### What it is

A playable world where:

- cartridges exist inside the space
- finished works return as objects
- tools are embedded in the environment
- the world remembers what happened

### Core logic connection

This is: artifact return, lineage, hub governance, world as connective tissue.

### Mechanics that matter

- object persistence
- cartridge as thing in space
- hidden workshop interactions
- world-state memory
- artifact transformation through play

### What makes it special

It is not a menu hub. It is a place that has been lived in.

### The respect barrier

If you want clean menus and clear progression: you'll bounce.

If you want a world you can inhabit and shape: you'll stay.

### What it opens

- true ecosystem feeling
- long-term engagement
- emotional attachment to artifacts
- cross-title continuity

### Why it's special

Tools, games, and memory all in one house.
